Instructions

Make your own Jukebox

1. Select the "Make your own JukeBox!" Button.

2. Enter what you want the song to be called into the text box.

3. Copy and paste the YouTube link of the song you want.

4. Add as many songs as you would like!

5. When you're done, select "All Done!".

6. From there, you can save the file or immediately play it (without saving!).

Using a Pre-Made JukeBox

1. Select the "Play a JukeBox!" Button.

2. You will be prompted to choose a .txt file that contains a saved Song List (it won't look like a list if you open it, but we promise it works).

3. Look over the Song List and make a choice of the song you want to listen to using the symbol/letter and number buttons.

4. Use the symbol/letter and number buttons to select your song.

5. At any time you can select a new song.

6. When you're done, just hit the "X" button to close the JukeBox.

Important Notes

- You must be connected to the internet to use the JukeBox.

- Only upload files that were created by the JukeBox application.

- You can use the spacebar to pause/resume while still operating in the application. **if you click out of the application after you have paused, you must make a new song selection to re-enable the pause/resume function.
